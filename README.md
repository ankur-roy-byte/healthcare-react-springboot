# Patient-SpringBoot-React

This project is simple hospital management system. 

### Using Tools & Technologies
``` 
- Spring Boot 2.2.2
- REST API (get, post, put, delete, patch)
- ModelMapper, DTO
- JPA, Hibernate
- React
- Bootstrap 4
- Oracle Database 11g
```
